HOW TO RUN:

1. open powershell as admin
2. go to GoldJotto directory
3. build using gradle command: gradle build
4. then run the jar file using this command: -jar build/libs/jotto-0.0.1-SNAPSHOT.jar
5. Open browser and go to local:8888 (local host)
